# Yoga by Luna - Website Clone

A faithful recreation of the Daily Mantra with Gaby yoga website, rebuilt as a modern static site.

## About the Original Site

This is a clone of https://dailymantrawithgaby.online/, a yoga and wellness website featuring:
- Yoga classes (Vinyasa, Hatha, Kundalini)
- Personal coaching and guided meditation
- Astro-yoga sessions
- Amazon retreat experiences
- Spiritual guidance by Luna Gabriela


1. Clone this repository
2. Open `index.html` in a web browser
3. For development, use a local server:
   ```bash

   # or
   npx serve .
   ```
